En primer lugar deberemos tener instalada la plataforma Home Assistant. 

Pasos para instalar HomeAsisstant -> https://www.home-assistant.io/installation/

Una vez tengamos insalado HomeAsistant, deberemos seguir los siguientes pasos para instalar el complemento Alba-Security

1. Descargar el contenido del repositorio

2. Modificar los valores de "options" en el archivo config.yaml

3. Mover la carpeta del proyecto al directorio \addons de Home Assistant

4. Instalar el complemento desde la tienda de complementos de Home Assistant
